# storybook-soft

storybook-soft is a component library with React, Typescript and Tailwind CSS.

```bash
  npm install storybook-soft

```

## Usage

- Add the `index.css` file to the root of your project.

```bash
   import 'storybook-soft/dist/index.css'
```
